# 📝 NGO Monthly Report Web App

A simple full-stack web app that allows NGOs to submit monthly reports and enables admins to view aggregated data for a selected month.

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Bootstrap 5
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (via Mongoose)
- dotenv
- CORS
